# Enhancelator

This is a fan made Emulator for "Milky way Idle" games Enhancing mechanic by (MangoFlavor).
Obviously i do not own the game nor any art assets in the this projects

## 🚀 快速启动 / Quick Start

### ⚡ 无需安装任何环境（推荐分享给他人）

**方法1：使用便携式服务器**（最简单，无需安装）
1. 下载 [Caddy](https://github.com/caddyserver/caddy/releases/download/v2.7.6/caddy_2.7.6_windows_amd64.zip) (约15MB)
2. 解压得到 `caddy.exe`，放到项目文件夹
3. 双击 `使用便携式服务器.bat`
4. 浏览器自动打开

📖 详细步骤请查看 `⚡快速开始-下载Caddy.txt`

### 💻 如果已安装 Python/Node.js

**方法2：Python版本**（推荐）
- 双击 `启动服务器.bat`

**方法3：Node.js版本**
- 双击 `启动服务器-NodeJS.bat`

### 🔍 不确定有什么环境？
- 双击 `环境检测.bat` 自动检测并给出建议

### 📖 详细说明
查看 `无需安装运行指南.txt` 获取完整的安装和使用说明

## ✨ 新增功能

### 双模式计算
- **工时费模式**: 根据工时费和利润率计算售价
- **成品售价模式**: 根据目标售价反推工时费

### 货币格式化
所有金币数字自动格式化为 K/M/B 单位，保留两位小数

### 智能高亮
- 🔵 蓝色：材料成本最低
- 🟢 绿色：
  - 工时费模式：总成本最低
  - 成品售价模式：工时费最高（最划算）
